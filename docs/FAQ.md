# FAQ

### How do I get an API key?
Sign up at OpenAI or Probex and copy your key into **AI Agent > Settings**.

### Can I use the plugin without an OpenAI key?
Yes. Enter a Probex key and base URL to use the alternate API.

### Why does the agent name change on refresh?
A random Australian name is chosen once per browser session to add personality.

### Does the chat work with caching plugins?
The widget loads after the page renders, so most caching plugins are compatible. Exclude the `wp-ai-agent` assets if problems occur.

### How do I clear stored conversations?
Go to **AI Agent > Chat History** and delete records, or adjust the retention setting to purge older chats automatically.
